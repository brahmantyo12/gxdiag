SETUP LAN SERVER

Mode LAN membutuhkan 1 komputer sebagai Server.

1. Install NodeJS di PC.
2. Buka Terminal/CMD, ketik:
npm install -g peer
3. Jalankan server:
peerjs --port 9000 --key peerjs --path /peerjs
4. Masukkan IP PC tersebut di sini.
